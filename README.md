# Actividad 3.1. Analizador léxico

**Fecha de entrega:** Jueves 23 de abril de 2026

## Propósito
Comprender la importancia del análisis léxico en el proceso de traducción de un lenguaje mediante el desarrollo de un analizador para el lenguaje C y un comparador de programas Python.

## Contenido del Repositorio
*   **Analizador Léxico C:** Implementado en `lexer_c.py`.
*   **Documentación:** Justificación del lenguaje y manual de usuario en la carpeta `Documentacion/`.
*   **Comparador Python:** (En desarrollo) Implementación de similitud de código.

## Evaluación
*   80% - Analizador de C y Comparador Python.
*   10% - Estándar de codificación.
*   10% - Manual de usuario.
